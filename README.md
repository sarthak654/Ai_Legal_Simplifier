# 🏛️ AI Legal Simplifier & Q&A Assistant

A privacy-first web application that simplifies legal documents using **100% local AI** - no external APIs, completely free to run.

## 🎯 What It Does

- **Extracts text** from PDF legal documents
- **Splits** documents into individual clauses  
- **Simplifies** complex legal language into plain English
- **Detects risk levels** (High/Medium/Low) in clauses
- **Answers questions** about the document using AI
- **Maintains privacy** - all processing happens locally

⚠️ **Disclaimer**: Educational tool only - not legal advice. Consult a lawyer for legal matters.

## 🛠️ Tech Stack

- **Backend**: Python, FastAPI, Ollama (local LLM), FAISS
- **Frontend**: React, Vite
- **AI**: gemma3:12b model running locally
- **Privacy**: 100% local processing, no external APIs
